Github倉儲分支 gh-pages 的用法(1)

第一種方法，直接在github網頁上新增名gh-pages的分支

再中間填入你要的分支名稱

Github倉儲分支 gh-pages 的用法(2)

第二種方法，再cmd下進行檢視、創造、切換分支

1. git branch 檢查目前分支數量

2. git branch gh-pages 建立一個名gh-pages分支

3. git checkout gh-pages 切換到gh-pages分支

git 語法複習與注意事項

1.git add -A 新增到git.

2.git commit -m "提交簡述"

3.git push (origin 分支名稱) 推送到github倉儲master，後面是推送到分支

4.git config--global user.email"使用著email"

5.git config--global user.name"使用著名稱"

注意!!4.和5可以到Y曹home下的".gitconfig.py進行修改"

利用 python -m http.server 執行近端檔案分享

4.指令python -m http.server 近端檔案分享

5.到網頁打上IPv4的位置和埠號(port XXXX)
